# 2D Virtual Labs – Simulation Based Learning Platform

A complete virtual laboratory platform designed for PSIT (Pranveer Singh Institute of Technology) to help students perform structured, interactive simulations across multiple engineering domains.

## Key Features

- *Interactive, simulation-based virtual labs*
- *Multi-subject support*, including:
  - Data Structures and Algorithms
  - Basic Electrical Engineering
  - Digital Electronics
  - Computer Architecture
  - Artificial Intelligence and Machine Learning
- *Structured learning workflow:*
  - Theory
  - Pre-Test
  - Simulation
  - Post-Test
  - Feedback
- *Lightweight, responsive, browser-based interface*

## Available Experiments

### Data Structures
- Sorting Algorithms (Bubble, Quick, Merge, Heap)
- Searching Techniques (Linear, Binary)
- Graph Algorithms (BFS, DFS, Dijkstra, Floyd–Warshall)
- Stack and Queue Implementations
- Tree Operations and Traversals
- Tower of Hanoi

### Basic Electrical Engineering
- Circuit Analysis
- Power Systems
- Electrical Machines
- Transformers

### Additional Subjects
- Computer Architecture
- Digital Electronics
- AI and ML Fundamentals

## Getting Started

### 1. Clone the repository

bash
git clone <repository-url>
cd 2D-Virtual-Labs


### 2. Run the project

Open the index.html file directly in a browser or use a local development server (e.g., Live Server in VS Code).

### 3. Use the platform

Navigate from subject → experiment → simulation.
