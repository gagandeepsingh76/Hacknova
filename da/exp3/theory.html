Perform Simple Linear Regression to Predict Salary Based on Experience

Simple Linear Regression is a statistical method used to model the relationship between two variables by fitting a linear equation to observed data.
In this experiment, we aim to predict an employee's salary based on their years of experience.

What is Simple Linear Regression?

It involves two variables:
- Independent Variable (X): Years of Experience
- Dependent Variable (Y): Salary

The relationship is modeled using a straight line:
Y = mX + c

Where:
- Y = Predicted Salary
- X = Years of Experience
- m = Slope of the line (how much salary changes per year of experience)
- c = Y-intercept (salary when experience is 0)

Steps Involved:

1. Import Necessary Libraries: Use libraries like pandas, numpy, matplotlib, and scikit-learn.

2. Load the Dataset: The dataset contains two columns: Experience and Salary.

3. Split Data into Training and Testing Sets: Training set is used to fit the model, and testing set is used to evaluate it.

4. Train the Model: Use LinearRegression from sklearn.linear_model to train the model.

5. Predict Results: Use the trained model to predict salaries for the testing data.

6. Evaluate the Model: Check metrics like Mean Squared Error (MSE) or R² Score to measure performance.

Basic Code Structure:
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, r2_score

# Load dataset
data = pd.read_csv('salary_data.csv')
X = data[['YearsExperience']]
Y = data['Salary']

# Split into train and test
X_train, X_test, Y_train, Y_test = train_test_split(X, Y, test_size=0.2, random_state=0)

# Train the model
model = LinearRegression()
model.fit(X_train, Y_train)

# Predict
Y_pred = model.predict(X_test)

# Visualize
plt.scatter(X_train, Y_train, color='blue')
plt.plot(X_train, model.predict(X_train), color='red')
plt.title('Salary vs Experience (Training set)')
plt.xlabel('Years of Experience')
plt.ylabel('Salary')
plt.show()

# Evaluation
print('Mean Squared Error:', mean_squared_error(Y_test, Y_pred))
print('R² Score:', r2_score(Y_test, Y_pred))


Conclusion:
Simple Linear Regression is a fundamental technique for predictive analysis. By training a model on the relationship between years of experience and salary, we can make salary predictions for new employees based on their experience level. Proper evaluation metrics ensure that the model is reliable for real-world use.
